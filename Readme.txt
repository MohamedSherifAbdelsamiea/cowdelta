This code compare 2 or more versions of excel sheets. 

steps:
1- Open a terminal and choose a directory in your PC to run the tool
2- clone the public repo --> git clone https://github.com/MohamedSherifAbdelsamiea/cowdelta.git
3- place in that folder 2 versions of COW tool output.
4- run the tool --> python3 COW_DELTA.py
